Steganography project


  • Steganography
  • Steganography is the practice of concealing information within another message or physical object to avoid detection
  • Steganography can be used to hide virtually any type of digital content, including text, image, video, or audio content

  • Steganography Project using python

Steps that you need to follow to make this project

  1. Import Modules.
  2. Create a Function to make a main frame
  3. Function to go back to the main frame
  4. Function to Encoding and Decoding frame.
  5. Create function for encoding image
  6. Create function for decoding image
  7. Function to decoding and generation of data
  8. Function to modify the pixels of image
  9. Function to enter the data pixels in image
  10. Function to enter hidden text
  11. GUI loop

Installing the nessecery modules to make this tool

1.Pyqt5 or tkinter

2.Pillow (PL)

3.io (allows us to manage the file-related input and output operations)

4.os

1.PIL module – This is the images module from the pillow. The PIL module helps to open, manipulate and save many different forms of images.

2.Import ImageTk – ImageTk module used to create and modify Tkinter photoimage from PIL images.

3.io import Bytesio – Bytes data in the memory.

4.Import os – This module is used for creating and removing any directory.

Then we need to create a main function

Idea about defining and buildin the code :

1.main() – Function for making the main frame.

2.root – Initializing the root window of Python Image Steganography Project.

3.title – Use to set title to window.

4.geometry – For setting dimensions of a window in pixels.

5.config – Used to configure attributes to the window, such as background color.

6.grid – for making grid frame on python image steganography project window

7.title.grid – Used to make a grid and here we are giving padding as 10 pixels. For the first row which is row equals to 1.

8.encode – encode variable creates the encode button using the command lambda: self and also gives padding as 14 and background color.

9.decode – decode variable creates the decode button using the command lambda: self and also gives padding as 14 and background color.

10.command lambda – It is used to pass data to the callback function.

11.grid_rowconfigure() – This method configures the row index of the grid and weight determines how large the row will occupy.

12..grid_columnconfigure() – This method configures the column index of the grid and weight determines how large the column will occupy.
